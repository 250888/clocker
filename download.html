前往 Releases 页面 查看源码
# 自行构建 Windows 版本
git clone https://github.com/250888/clocker.git
cd clocker/clocker_app
flutter build windows --release
# 输出: build/windows/x64/runner/Release/

Android 即将发布

支持 Android 6.0+，包含摄像头注意力检测、白噪音、专注计时

前往 Releases 页面 查看源码
# 自行构建 Android 版本
git clone https://github.com/250888/clocker.git
cd clocker/clocker_app
flutter build apk --release
# 输出: build/app/outputs/flutter-apk/app-release.apk
